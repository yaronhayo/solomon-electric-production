## Quick Answers: Electrical Installation FAQ

### How do I know what size electrical service I need?

**We calculate based on**:
- Home square footage
- Number and size of AC units
- Kitchen appliances (electric range, ovens)
- Pool/spa equipment
- EV chargers (each needs 40-50 amps)
- Workshop or home gym equipment
- Future expansion plans

**Typical Miami homes**:
- 1,500-2,500 sq ft: 200 amps
- 2,500-4,000 sq ft: 200-400 amps
- 4,000+ sq ft or multiple large loads: 400 amps

### Should I future-proof my electrical system?

**Absolutely.** Today's luxury is tomorrow's standard:
- EV chargers (will you buy electric vehicles?)
- Solar panels + battery storage
- Home automation systems
- Home offices (more circuits, dedicated internet)
- Outdoor kitchens and living spaces

**Adding capacity later is expensive.** We design systems with 20-30% growth capacity.

### What's the difference between rough-in and trim-out?

**Rough-in** (before drywall):
- Install electrical panel
- Run all wiring in walls/ceiling/attic
- Install electrical boxes for outlets/switches
- Rough-in inspection before drywall closes walls

**Trim-out** (after drywall, painting):
- Install outlets, switches, fixtures
- Connect panel circuits
- Install smoke detectors
- Test all systems
- Final inspection

## Complete Electrical System Components

### Service Entrance & Panel

**Main electrical service**:
- Weather head and service drop
- Meter base (FPL-approved)
- Main electrical panel (200A or 400A)
- Grounding system (ground rods, bonding)
- Whole-home surge protection

**Sub-panels** (if needed):
- Detached garage/workshop
- Pool equipment
- Guest houses
- Second-floor circuits (large homes)

### Branch Circuit Wiring

**General circuits**:
- Lighting circuits (15A)
- General outlet circuits (15-20A)
- Small appliance kitchen circuits (20A, two required)
- Bathroom circuits (20A)
- Laundry circuit (20A)

**Dedicated circuits**:
- Refrigerator (20A)
- Microwave (20A)
- Dishwasher (15-20A)
- Disposal (15-20A)
- Each bathroom (20A)
- Electric range (40-50A, 240V)
- AC units (varies by tonnage)
- Pool pump (20-30A)
- EV charger (40-60A, 240V)

### GFCI & AFCI Protection

**Code requires**:
- GFCI: bathrooms, kitchen countertops, garages, outdoors, laundry
- AFCI: bedrooms, living rooms, dining rooms, family rooms, dens

**Implementation options**:
- GFCI/AFCI outlets at each location
- GFCI/AFCI breakers protecting entire circuits
- Combination AFCI/GFCI breakers (latest code)

## New Construction Electrical Process

### Phase 1: Planning & Design

**Pre-construction services**:
- Review architectural plans
- Perform load calculations
- Design electrical system layout
- Plan panel locations and sizes
- Coordinate smart home pre-wiring
- Submit permit applications

### Phase 2: Rough-In Installation

**Before drywall**:
- Install temporary power for construction
- Set main panel and sub-panels
- Drill holes for wire runs
- Run all circuit wiring
- Install electrical boxes (outlets, switches, fixtures)
- Wire HVAC disconnects
- Install conduit for underground (pool, exterior)
- First inspection (rough-in approval)

### Phase 3: Trim-Out Installation

**After drywall and painting**:
- Install all outlets and switches
- Install light fixtures and ceiling fans
- Connect appliances and equipment
- Install smoke and CO detectors
- Test all circuits and devices
- Label panel clearly
- Second inspection (final approval)

### Phase 4: Final Commissioning

**System activation**:
- FPL service installation
- Power-up and testing
- Smart home programming
- Homeowner training
